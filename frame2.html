HTML is currently a markup language that helps in building webpage through various tags and elements. It provides overall framework about how a website looks when we open it through a web browser. CONCEPT

Html is a markup language that is used to create websites. It is used to identify various components of a document like headings, paragraphs, lists etc. It defines the content and structure of a website.

So, what is the meaning of hypertext markup language?

HyperText- Pieces of text that can link to other documents!

Markup Language- It indicates that it uses a language type called markup which uses html tags wihtin it.

TAGS

HTML uses enormous amount of tags within it to make a web page.
So, firstly what are these tags?
Tags are nothing but a combination of grater than and less than symbol. We write the required or specified property within those '<' & '>' to give some functionality to it.

For ex,

<p> PARAGRAPH </p>

This specifies that a 'p' tag is created which is called a paragraph so that we can write our list of paragraphs within it!

Just like this, there are many other tags used... Let's look at the table for many such tags!

  • Unpaired/Standalone tags

HTML STRUCTURE/ BOILERPLATE

HTML has a particular structure in it's code. We have a specific boilerplate in HTML5.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8"/>
<title>TITLE</title>
<link rel="stylesheet" href="S:\xyz\html website\css_1.css" />
</head>
<body>
</body>
</html>

<html lang="en">    ------->html tag which defines that it is written in english language.
<head>    -------> 1st part of html is head section where we write title, meta tags etc.
<meta charset="utf-8"/>    -------> This ensures that characters are displayed correctly.
<title>TITLE</title>    -------> The required title in the title bar of the browser is displayed.
<link rel="stylesheet" href="S:\xyz\html website\css_1.css" />    -------> We generally use this tag to link a html page to a css page which will be discussed later.
</head>    -------> Head section f the html file is ended with this closing tag.
<body>    -------> This is the body of the html file.
</body>
</html>

LISTS

List is a property used in HTML to create a set of elements in an arrangement/ order.
In HTML, lists are used for different purposes in different types.

There are three types of lists:
Ordered, Unordered & Description lists

These lists are represented as <ol> </ol>, <ul> </ul> & <dl> </dl> respectively.
Also, there is a tag for the elements that we insert in a list. They're repsresented as
<li> list item name </li>

Ordered:These are the type of list which have specific mentioning of the list element.
Eg: Numberings like 1 2 3..
Syntax:
<ol>
<li> 1st list item name </li>
<li> 2nd list item name </li>
</ol>

O/P:
1. 1st list item name
2. 2nd list item name

Unordered:These are the type of list which doesn't have specific mentioning of the list element.
Syntax:
<ul>
<li> list item name </li>
<li> list item name </li>
</ul>

O/P:

  • list item name
  • list item name

Description:These are the type of list which have two elements called description term and description definition.
Syntax:
<dl>
<dt> Animals </dt>
<dd> Dog </dd> <dd> Cat </dd>
</dl>

O/P:

Animals
Dog
Cat

Cell Padding & Cell Spacing in tables

Usually, these properties are used in tables for the cell styling.

Cell padding: It defines the amount of space between the cell border and its content (borders are discussed in CSS).
Cell spacing: Amount of space between the cells is described by cell spacing, we can modify it through this tag used in style element.

Also, height, width of the table is described by their names itself in many units like px(pixels), cm(centimeters) etc.. and Align property is used to align the cells in either of the way i.e., left or right.
